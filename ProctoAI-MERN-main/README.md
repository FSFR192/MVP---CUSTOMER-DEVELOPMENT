# ProctoAI-CUSTOMER DEVELOPMENT

ProctoAI es un Sistema de Supervisión de Exámenes Automatizado (AEPS) desarrollado con algoritmos de Inteligencia Artificial (IA) de vanguardia para la realización de pruebas en línea.
Este sistema integral está diseñado para garantizar la integridad y la seguridad de las evaluaciones en línea. El proyecto aprovecha tecnologías clave como React.js, Redux, Node.js y TensorFlow.js para ofrecer una solución de supervisión de exámenes rica en funcionalidades.
